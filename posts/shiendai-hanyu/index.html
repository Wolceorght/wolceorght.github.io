《現代漢語》要點筆記（上）

發表於 2025年5月26日 分類於 枯筆居

緒論

現代漢語概說

什麼是現代漢語、現代漢民族共同語

現代漢語是現代漢民族所使用的語言，包括多種方言和民族共同語。現代漢民族共同語是以北京話為標準音，以北方話為基礎方言，以典範的現代白話文著作為語法規範的普通話。

七大方言分佈區

  1. 北方方言：以北京話為代表，是分佈地域最廣的方言。可分為四個次方言：華北 — 東北方言、西北方言、西南方言、江淮方言。
  2. 吳方言：以蘇州話為代表。
  3. 湘方言：以長沙話為代表。
  4. 贛方言：以南昌話為代表。
  5. 客家方言：以廣東梅縣話為代表，分佈分散但內部差異不大。
  6. 閩方言：可分為閩東、閩南、閩北、閩中、莆仙五個次方言。閩東方言以福州話為代表，閩南方言以廈門話為代表。
  7. 粵方言：以廣州話為代表。

現代漢語的特點

屬於漢藏語系。

語音方面
  1. 沒有複輔音
  2. 元音佔優勢
  3. 音節整齊簡潔
  4. 有聲調
詞彙方面
  1. 單音節語素多，雙音節詞佔優勢
  2. 構詞廣泛運用詞根複合法
  3. 同音語素多
語法方面
  1. 主要用語序和虛詞（而非形態）表示語法意義
  2. 詞、短語、句子的結構規則基本一致
  3. 詞類和句法成分關係複雜
  4. 量詞和語氣詞十分豐富

第一章 語音

第一節 語音概說

語音是語義的表達形式，語言的物質外殼。

語音的性質

語音的物理屬性
  1. 音高：聲音高低，取決於發音體振動的快慢
  2. 音強：聲音強弱，取決於發音體振幅的大小
  3. 音長：聲音長短，取決於發音體振動時間的久暫
  4. 音色：聲音特色，取決於音波波紋曲折形式不同
語音的生理屬性

發音器官可分為以下部分：

  1. 呼吸器官
  2. 喉頭和聲帶
  3. 咽腔、鼻腔和口腔
語音的社會屬性
  1. 語音和意義並沒有必然聯繫，而是約定俗成
  2. 語音具有系統性

語音單位

音素

音素是最小的語音單位。（從音色角度劃分）

輔音：氣流經過口腔或咽頭受阻礙而形成的音素

元音：氣流振動發出聲音，經過口腔、咽頭不受阻礙而形成的音素

音節

由音素構成，是聽話時自然感到的最小的語音單位。

聲母、韻母、聲調

是漢語音韻學的傳統字音分析方法。

聲母：位於音節前段，主要由輔音構成

韻母：位於音節後段，由元音或元音加輔音構成

聲調：依附在聲韻結構中具有區別意義作用的音高型式

音位

音位是一個語音系統中能夠區別意義的最小的語音單位。（從社會屬性角度劃分）

記音符號

  1. 漢字記音：直音法、反切法
  2. 注音符號
  3. 羅馬字拼音字母：威妥瑪方案、國語羅馬字、漢語拼音方案

第二節 輔音與聲母

輔音的發音部位

發音時氣流受阻的位置叫做發音部位。普通話輔音可據此分為七類：

  1. 雙脣音 (b, p, m)
  2. 脣齒音 (f)
  3. 舌尖前音 (z, c, s)
  4. 舌尖中音 (d, t, n, l)
  5. 舌尖後音 (zh, ch, sh, r)
  6. 舌面前音 / 舌面音 (j, q, x)
  7. 舌面後音 / 舌根音 (g, k, ng, h)

輔音的發音方法

發音時喉頭、口腔和鼻腔節制氣流的方式和狀況叫做發音方法。

  1. 根據阻礙方式劃分
    1. 塞音 (b, p, d, t, g, k)
    2. 擦音 (f, h, x, sh, r, s)
    3. 塞擦音 (j, q, zh, ch, z, c)
    4. 鼻音 (m, n, ng)
    5. 邊音 (l)
  2. 根據聲帶是否振動劃分
    1. 帶音 / 濁音 (m, n, ng, l, r)
    2. 不帶音 / 清音 (其餘)
  3. 根據氣流強弱劃分（塞音、塞擦音）
    1. 送氣音 (p, t, k, q, ch, c)
    2. 不送氣音 (b, d, g, j, zh, z)

第三節 元音與韻母

元音的發音

單元音的發音

單元音是發音時口形始終不變的元音。

  1. 根據舌位高低劃分：高、半高、半低、低
  2. 根據舌位前後劃分：前、央、後
  3. 根據脣形圓展劃分：圓脣、展脣 / 不圓脣
  4. 舌面、舌尖、捲舌劃分：
    1. 舌面元音 (a, o, e, ê, i, u, ü)
    2. 舌尖元音 (舌尖前 - i, 舌尖後 - i)
    3. 捲舌元音 (er)
複元音的發音

複元音是發音時舌位、脣形都有變化的元音。

  1. 前響複元音 (ai, ei, ao, ou)
  2. 後響複元音 (ia, ie, ua, uo, üe)
  3. 中響複元音 (iao, iou, uai, uei)

韻母的發音

  1. 開口呼：韻母開頭不是 i, u, ü 的韻母
  2. 齊齒呼：韻母開頭是 i 的韻母
  3. 合口呼：韻母開頭是 u 的韻母
  4. 撮口呼：韻母開頭是 ü 的韻母

韻母的結構

韻母可分為韻頭 / 介音 (i, u, ü)、韻腹、韻尾 (n, ng, i, u)。

第四節 聲調

聲調是依附在音節上的超音段成分，主要由音高構成。

調值：依附在音節裏高低升降的音高變化的固定格式，也就是聲調的實際讀音。

調類：把調值相同的字歸納在一起所建立的類。

普通話的聲調

  1. 陰平 (55)
  2. 陽平 (35)
  3. 上聲 (214)
  4. 去聲 (51)

古今調類和四聲平仄

古今調類

古清聲母平聲→陰平

古濁聲母平聲→陽平

古清聲母、次濁聲母上聲→上聲

古去聲、全濁聲母→上聲

入派四聲：古清聲母入聲→四聲；古次濁聲母入聲→去聲；古全濁聲母入聲→陽平

平仄

平→平聲

仄→上聲、去聲、入聲

第五節 音節

普通話音節結構的特點

  1. 一個音節使用一至四個音素符號拼寫
  2. 元音在音節中佔優勢
  3. 音節中可以沒有輔音
  4. 漢語音節都有聲調、韻腹

普通話的聲韻拼合規律

  1. 雙脣音和舌尖中音 d, t 能拼開、齊、合（雙脣限於 u），不能拼撮
  2. 脣齒音、舌面後音、舌尖前音、舌尖後音能拼合開、合（脣齒限於 u），不能拼齊、撮
  3. 舌面前音能拼齊、撮，不能拼開、合
  4. 舌尖中音 n, l 和零聲母能拼四呼

第六節 音變

變調

  1. 上聲變調：上連上，前字讀 35；上連非上，前字讀 21；句末上，讀 214。
  2. 「一、不」變調：去聲前讀 35，非去聲前讀 51，句末讀原調。

輕聲

輕聲是四聲在一定條件下變成比原調又輕又短的聲調變體。

輕聲和音強、音長、音高、音色都有關係。

作用：區分詞義、詞性，增強語言節奏感。

兒化

兒化是一個音節中，韻母帶上捲舌色彩的一種特殊音變現象。

作用：區分詞義、詞性，表現細小、輕鬆或表示親切、喜愛的感情色彩。

第七節 音位

音位變體

一個音位往往包含一些不同的音，這些音就是這個音位的音位變體。

條件變體：在一定條件下出現的音位變體

自由變體：可以自由替換的音位變體

歸納音位的標準

可以造成意義不同的語音差異就叫音位對立，構成這種差異的語音特徵就叫區別特徵。

音位的不同條件變體各有自己的分佈條件，絕不出現在相同的位置上，這叫做互補分佈。

互補分佈一般不造成音位對立，但同時應考慮當地人的音感差異。

第八節 語調

語調是說話時句子的停頓、聲音的輕重快慢和高低長短的變化。

停頓

說話時，段落之間、語句中間和末尾出現的聲音間歇。

意羣：詞與詞結合在一起構成的意義整體。

節拍羣 / 音步：在較大的意羣裏根據意義關係和結構層次劃分出的更小的意義整體。

停頓大體出現在意羣和節拍羣後面。

重音

  1. 語法重音：句子中某些語法成分需要重讀
  2. 邏輯重音：句子中某些需要突出或強調的詞語需要重讀

句調

句調是指整句話的聲音高低升降的格式。

句調和字調（聲調）是全局與局部的關係。

句調的四種形式
  1. 升調：反問、疑問、驚異、號召
  2. 降調：陳述、感歎、請求
  3. 平調：嚴肅、冷淡、敘述
  4. 曲調：含蓄、諷刺、意在言外

第二章 文字

第一節 漢字概說

文字是記錄語言的書寫符號系統，是最重要的輔助性交際工具。

漢字的特點

  1. 漢字是表意文字
  2. 漢字是形體複雜的方塊結構
  3. 漢字分化同音詞能力強
  4. 漢字有超時空性

第二節 漢字的形體

現行漢字的前身

  1. 甲骨文：殷商時代刻在龜甲、獸骨上的文字
  2. 金文：西周青銅器上的文字
  3. 篆書
    1. 大篆：春秋戰國時代的秦國文字
    2. 小篆：秦始皇統一六國後整理推行的標準字體
  4. 隸書：產生於秦代（秦隸），於漢代通行（漢隸）
  5. 楷書：興於漢末，盛行於魏、晉，一直沿用至今
  6. 輔助性字體
    1. 草書：章草、今草、狂草
    2. 行書

現行漢字的形體

  1. 手寫體：楷書、行書
  2. 印刷體：宋體、仿宋體、楷體、黑體

第三節 漢字的結構

結構單位

  1. 筆畫：構成漢字的最小單位
  2. 部件：由筆畫構成的具有組配漢字功能的構字單位
  3. 整字：可分爲獨體字和合體字

部首是字書中各部領頭的部件或筆畫，具有字形歸類作用。

造字法

  1. 象形：用描繪事物形狀來表示字義
  2. 指事：用象徵性符號或在象形字上添加提示符號來表示字義
  3. 會意：用兩個或以上部件合成一個字，把這些部件的意義合成新字字義
  4. 形聲：由表示字義類屬的部件（形旁）和表示字音的部件（聲旁）組成新字

第三章 詞彙

第一節 詞彙概說

詞彙是一種語言裏所有的（或特定範圍內的）詞和固定短語的總和。

詞彙單位

語素

語素是最小的音義結合的語言單位。

  1. 成詞語素（詞根）：能夠獨立成詞的語素

  2. 不成詞語素：不能單獨成詞的語素

    1. 不定位不成詞語素（詞根）
    2. 定位不成詞語素（詞綴）

詞根：表示詞的基本意義

詞綴：表示詞的附加意義、起語法作用

詞

詞是語言中最小的能獨立運用的音義結合的語言單位。

短語是由詞逐層組成的、沒有語調的語言單位。

固定短語

固定短語是詞與詞的固定組合，一般不能任意增減、改換其中成分。

固定短語分為：

  1. 專名
  2. 熟語
    1. 成語
    2. 慣用語
    3. 歇後語
    4. 諺語
  3. 縮略語
    1. 簡稱：較複雜名稱的簡化形式
    2. 數詞略語：使用數字對並列詞語的語素或語義成分概括出來的略語（如三峽、四化）

詞的結構類型

單純詞

單純詞是由一個語素構成的詞。

  1. 聯綿詞：兩個不同的音節連綴成一個語素，表示一個意義

    雙聲詞、疊韻詞

  2. 疊音詞：由不成語素的音節重疊構成

  3. 音譯外來詞

合成詞

合成詞是由兩個或以上的語素構成的詞。

合成詞有以下三種構成方式：

  1. 複合式：兩個或以上不相同的詞根結合
    1. 聯合型：兩個意義相同、相近、相關或相反的詞根並列組合
    2. 偏正型：前一詞根修飾、限制後一詞根
    3. 補充型：後一詞根補充說明前一詞根
    4. 動賓型：前一詞根表示動作，後一詞根表示動作、行為所支配關涉的事物
    5. 主謂型：前一詞根表示被陳述的事物，後一詞根陳述前一詞根
  2. 重疊式：由相同的詞根語素重疊而成
  3. 附加式：由詞根和詞綴構成
    1. 前加型（前綴 + 詞根）：老 -、小 -、第 -、阿 -……
    2. 後加型（詞根 + 後綴）：- 子、- 頭、- 兒、- 性、- 者、- 化、- 於……

第二節 詞義及其性質和構成

詞義的性質

  1. 概括性
  2. 模糊性
  3. 民族性

詞義的構成

概念義

詞義中同表達概念有關的意義部分叫作概念義。

色彩義

色彩義附著在詞的概念義之上，表達人或語境賦予的特定感受。

  1. 感情色彩
  2. 語體色彩
  3. 形象色彩

第三節 義項和義素

義項

義項是詞的理性意義的分項說明。

一個詞的每個義項存在互補關係。

義項的分類
  1. 基本義：最常用的意義
  2. 轉義：由基本義直接或間接地轉化而來
    1. 引申義：在基本義的基礎上經過推演發展而產生的意義
    2. 比喻義：借用一個詞的基本義來比喻另一事物，所產生並被固定下來的新意義

義素

義素是構成詞義的最小意義單位。

義素分析

  1. 明確分析對象

    義素分析在一些相關的詞（同一語義場）中進行。

  2. 找出所選定的詞的共同義素與區別義素。

  3. 義素確定後，採取方法進行表達。

作用：幫助準確掌握、解釋、理解語義，突出顯示詞義之間的異同和聯繫。

第四節 語義場

語義場是既有共同義素又有區別義素的一組詞的相關語義的聚合。

語義場有不同的層次，上一層次中某個詞的義素必然為下一層次的各詞所具有，而下一層次又必然有自己的特殊的義素。

語義場分析

  1. 類屬義場：成員同屬一個較大的類
  2. 順序義場：成員按照某種固定順序排列
  3. 關係義場：一般由兩個成員組成，二者處於某種關係的兩端
  4. 同義義場：由意義相同或相近的詞的相關語義組成
  5. 反義義場：由意義相反或相對的兩個詞的相關語義組成

同義詞的差別

理性方面
  1. 意義的輕重
  2. 範圍的大小
  3. 集體和個體不同
  4. 搭配對象不同
色彩方面的差別
  1. 感情色彩不同
  2. 語體色彩不同
  3. 詞性方面的差異

同義詞的作用

  1. 使語言的表達精確嚴密
  2. 使語體風格鮮明
  3. 使文句生動活潑，賦予變化
  4. 使表達的語氣委婉
  5. 連用使語氣加強，語意完足

反義義場的類型

  1. 互補反義義場 (A=-B, B=-A; -B=A, -A=B)
  2. 極性反義義場 (A=-B, B=-A; -B≠A, -A≠B)

反義詞的作用

  1. 揭示事物的矛盾，深刻地表示事物的特點
  2. 連用使語氣加強，強調核心意思
  3. 構成對偶、映襯句，使語言深刻有力
  4. 臨時反義詞使語言新穎而又簡練（如舊聞）
  5. 可以用來構成合成詞

第五節 詞義和語境的關係

語境就是語言單位出現時的環境。

語言語境：口語中的前言後語、書面語中的上下文

情景語境：說話時牽涉的人或物、時間處所、社會環境以及輔助性交際手段（包括表情、姿態、手勢等非語言因素）

語境對詞義的影響

  1. 語境使詞義單一化
  2. 語境使詞義具體化
  3. 語境增加臨時性意義
  4. 語境表現出詞義的選擇性

第六節 現代漢語詞彙的組成

基本詞彙和一般詞彙

基本詞彙

基本詞彙是詞彙中最主要的部分。

基本詞彙的特點：

  1. 穩固性
  2. 能產性
  3. 全民常用性
一般詞彙

基本詞彙以外的詞彙是一般詞彙。

現代漢語中，一般詞彙包括以下分類：

  1. 古語詞：現代漢語仍然使用的那部分古代漢語詞語
    文言詞、歷史詞

  2. 方言詞：普通話從各方言中吸取來的詞

  3. 外來詞 / 借詞：從外族語言借來的詞

    1. 音譯

    2. 音義兼譯

    3. 音譯加漢語語素

    4. 借形

      字母式借形詞、漢字式借形詞（日語）

  4. 行業語：各種行業應用的專有詞語

  5. 隱語：個別社會集團或祕密組織內部懂得並使用的特殊用語

第七節 熟語

熟語是人們常用的定型化了的固定短語。

成語

成語是一種相沿習用、含義豐富、具有書面語色彩的固定短語。

成語的特徵
  1. 意義整體性
  2. 結構凝固性
  3. 風格典雅性
成語的來源
  1. 神話寓言
  2. 歷史故事
  3. 詩文語句
  4. 口頭俗語
成語的結構
  1. 聯合結構
  2. 偏正結構
  3. 動賓結構
  4. 補充結構
  5. 主謂結構

諺語

諺語是羣衆口語中通俗精煉、含義深刻的固定語句。

慣用語

慣用語是口語中短小定型的習用的短語，大多是三字格的動賓短語。

歇後語

由近似於謎面、謎底的前後兩部分組成的帶有隱語性質的口頭固定短語。

分為喻意歇後語、諧音歇後語。
